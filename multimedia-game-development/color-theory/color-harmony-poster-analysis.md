# 色彩調和與海報配色分析筆記

> 適用課程／領域：明新科技大學 多媒體遊戲發展系相關學習資料  
> 主題：色彩學、視覺設計、海報分析、主色／輔色／強調色比例

## 一、四種色彩調和概念重點

### 1. 秩序性調和
以規律、重複、漸層或有系統的方式安排色彩，讓畫面產生穩定、整齊、有節奏的感覺。常見於色票排列、系列化設計、規則構圖。

### 2. 親和性調和
畫面中的顏色彼此有共同特質，例如共同偏暖、共同偏冷、共同低彩度，或角色、背景、文字之間有相同色感，因此看起來自然、舒服、不突兀。

### 3. 類似性調和
使用色相環上相鄰或接近的顏色，例如藍、藍綠、青綠，或黃、橘、紅橘。這種方法能讓畫面統一、有一致氣氛。

### 4. 明瞭性調和
利用強烈對比讓重點清楚，例如黑底紅字、冷暖對比、明暗對比、高彩度與低彩度對比。常用於標題、警告、焦點或視覺導引。

## 三、單張海報重點整理

### 1. 超級瑪利歐銀河電影版
- **調和判斷**：類似性調和為主，明瞭性調和為輔。
- **主色 70%：深藍／藍色系**  
  用於夜空、海面、背景，建立夢幻、冒險、宇宙與海洋氣氛。
- **輔色 20%：洋紅／粉紅色**  
  用於天空漸層、標題、煙火光點，增加活潑、童趣與節慶感。
- **強調色 10%：黃色／金黃色**  
  用於船燈、星光、標題亮部，面積小但視覺吸引力強。
- **設計重點**：藍色系統一氛圍，黃與洋紅負責製造亮點與熱鬧感。

### 2. 狸想世界
- **調和判斷**：類似性調和為主，親和性調和為輔。
- **主色 45%：黃色／金黃色**  
  用於背景與標題，營造明亮、歡樂、童趣感。
- **輔色 35%：橘色／橘紅色**  
  用於角色毛色、背景漸層與中段畫面，加強溫暖與活力。
- **強調色 20%：深紅棕／暗棕色**  
  用於熊的毛色與陰影，讓畫面有重量感與層次。
- **設計重點**：黃、橘、紅棕都屬於暖色範圍，整體協調且有喜劇感。

### 3. 曼達洛人與古古
- **調和判斷**：類似性調和為主，明瞭性調和為輔。
- **主色 50%：黃橘色系**  
  用於背景、光線與沙漠感天空，形成冒險、炎熱、科幻西部感。
- **輔色 35%：橘紅／紅橘色**  
  用於爆炸、戰鬥場景、火光與下方暗部，增加戲劇張力。
- **強調色 15%：米白＋青綠標題色**  
  米白用於亮部與背景緩衝，青綠標題與暖色背景形成冷暖對比。
- **設計重點**：暖色建立世界觀，青綠標題讓主題文字跳出。

### 4. 鬼上身
- **調和判斷**：明瞭性調和為主，類似性調和為輔。
- **主色 75%：黑色／深藍黑色**  
  用於森林、夜晚背景與暗部，製造恐怖、未知、壓迫感。
- **輔色 15%：灰藍色**  
  用於道路、車燈照亮區、草叢與霧氣，增加冷光層次。
- **強調色 10%：紅色**  
  用於片名與中央文字，形成危險、血腥、警告的視覺焦點。
- **設計重點**：黑暗背景與紅色文字的強對比，讓片名非常明確。

### 5. 花綠青綻放之時
- **調和判斷**：類似性調和為主，親和性調和為輔。
- **主色 50%：淺藍／藍色系**  
  用於背景、水氣、亮部與天空感，營造清新、青春、透明感。
- **輔色 30%：中藍色**  
  用於人物衣服、陰影與中段背景，增加冷靜與層次。
- **強調色 20%：藍綠／青綠色**  
  用於人物頭髮、輪廓與暗部，建立焦點與深度。
- **設計重點**：藍、藍綠、青綠相近，畫面統一、安靜且帶有水感。

## 四、學習應用筆記

### 1. 判斷主色、輔色、強調色的方法
- **主色**：畫面面積最大，決定整體氣氛。
- **輔色**：用來補充主色，增加層次與畫面豐富度。
- **強調色**：面積通常較小，但最吸睛，常用於標題、光源、角色焦點或重要資訊。

### 2. 做海報或遊戲視覺時的應用
- 想要畫面統一：可使用**類似性調和**。
- 想要畫面自然舒服：可使用**親和性調和**。
- 想要重點清楚、標題跳出：可使用**明瞭性調和**。
- 想要系列感或規則感：可使用**秩序性調和**。

### 3. 多媒體遊戲發展系可應用方向
- **角色設計**：用主色決定角色性格，用強調色凸顯記憶點。
- **場景設計**：用類似色建立世界觀，例如海洋關卡用藍綠系、沙漠關卡用黃橘系。
- **UI 設計**：主色建立介面風格，強調色用於按鈕、警告、任務提示。
- **海報與宣傳圖**：以主色建立情緒，輔色增加層次，強調色抓住觀眾視線。

## 五、簡短結論

這五張海報大多使用**類似性調和**建立整體統一感，再依照作品類型加入不同的強調色。動畫與冒險類作品常用高彩度暖色或亮色製造活力；恐怖類作品則常用黑色作為主色，再用紅色形成強烈明瞭對比。這些分析可以作為日後製作遊戲海報、角色視覺、場景配色與 UI 色彩配置的參考。
